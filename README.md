# Проектная работа 11 спринта — «Будущее 2.0»

## Содержание

- [Task1 — Анализ текущего состояния и целевая архитектура](./Task1/README.md)  
  C4 Container диаграмма целевой архитектуры, анализ проблемных мест системы, приоритизация по методу MoSCoW.

- [Task2 — Доменная декомпозиция системы](./Task2/README.md)  
  Разделение на 5 бизнес-доменов, Data Flow Diagram с потоками данных между доменами, обоснование архитектурных решений.

- [Task3 — Технологическая трансформация](./Task3/README.md)  
  Технический радар (ADOPT/TRIAL/ASSESS/HOLD), роадмап трансформации на 4 квартала, обоснование этапов.

## Итог

Репозиторий содержит комплексное решение по трансформации монолитного DWH в событийно-ориентированную архитектуру с доменной декомпозицией. Решение включает анализ проблем, целевую архитектуру, план миграции и обоснование технологических изменений для создания масштабируемой системы с self-service витриной данных.
